c++实现经典数据结构
Posted on 2020-11-04   Updated on 2020-11-14 algorithm c++

Data Structure(c++)

因为确定这次秋招是去到鹅厂，微信的技术栈还是以c++为主，所以最近也是一直在学习c++，俗话说得好实践是检验真理的唯一标准。c++这门语言本身的特点就是语法十分繁杂，整个语言包含的东西十分的广泛，现在很多的语言都是以精简为自己的语言理念，程序员可以很好地上手，而不是陷入到学习语言语法的误区中去。c++确实走了一条与上述理念相反的道路，c++是一门野心非常大的语言，它几乎是想要囊括一切，你想到或者是想不到的它都有！

单纯的学习语法是一件很枯燥的事情，所以我会上手一些实践操作。之前在19年寒假的时候无意中发现了 https://www.coursera.org/learn/algorithms-part1 这门数据结构与算法课程，Robert Sedgewick教授是我见过的把数据结构和相关算法讲的最清楚的老师，课程中的代码都是非常优雅的实现，而且相关讲义制作十分精良，如果有兴趣还可以去买一本教授出版的算法书籍，实在是令人受益匪浅！

不过Robert教授的代码都是用Java实现的，于是我在自己学习的基础上用c++实现相关的数据结构和算法，并且完善视频中一些没有给出的细节！

1.Binary Search Tree

搜索二叉树是一种常见的数据结构，主要的特点就是二叉树上的每一个节点的值要小于左边子树的所有值，并且大于右边子树的所有值，中序遍历就可以顺序输出所有的节点值！

API：

  • void put(K key, V val)
  • const V* get(K key)
  • const K* get_min_key()
  • const K* get_max_key()
  • void del(K key)
  • void del_min()
  • void del_max()
  • const K* floor(K key)
  • const K* celling(K key)
  • int rank(K key)
  • int size()
  • int level()
  • void print()

4.Priority Queue

优先队列能让我们根据特定的优先级，在一组元素中拿出优先级最高或者是最低的元素。

优先队列有很多中实现方法，这里用heap实现，这里的堆用树表示，一个节点的元素不会小于它的孩子节点，不同于之前的创建一个专用于树的节点用指针表示孩子节点，这里直接用数组表示！

数组从1开始索引，具体有以下好处：

  • 对于idx = k，父节点是k / 2，子节点是2 * k和2 * k + 1
  • 对于root节点很好管理，可以根据k / 2 = 0来判断是不是root节点

API:

  • int size()
  • void insert(int)
  • int max()
  • int del_max()
