服务器上用docker部署静态网站
Posted on 2019-08-24   Updated on 2020-11-08 docker nginx

在阿里云上用docker部署静态网站

docker这种先进的容器技术为我们如今的开发、测试和部署都带来了极大的方便，我们不用再苦恼于开发环境、测试环境和部署环境的不一致了。docker用轻量化的容器技术为我们带来了一种学习成本极低的全新开发体验。

我也是最近开始接触docker，今天看了一个用docker部署静态网站的视频，故把这个写下来，加强印象。

在以往的经验中，我们要在服务器上部署静态网站就必须要安装nginx，然后部署，但是想一想我们的系统只能有一个nginx的版本，如果还需要其他例如apache，java等环境，要保持环境的一致性是几乎不太现实的，难道每次都是重装吗？

有了docker的容器隔离技术，我们就要换一个思路。

首先我们要拉取一个centos镜像：docker pull docker.io/.centos:latest

有了这个镜像我们就可以启动一个容器：docker run -i -t i
